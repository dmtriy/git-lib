
класс Route - разбор строки url из $_SERVER['REQUEST_URI'] . строка url должна быть сформирована :

контролер/модель/парам1/дата1/парам2/дата2 и т.д

контролер = возможно модель = по умолчанию.Параметров нет
контролер/модель = возможно.Параметров нет
контролер/парам1/дата1/парам2/дата2 = нельзя.При наличии параметров модель должна быть



 класс Db - подключение к базе данных MySql используя PDO на основе патерна "одиночка".

статический setConfig($arrayConfig) - установка конфигурации подключения вход массив с полями name,user,host,pass.
  если массив $arrayConfig не определен или не все значения присутствуют то по умолчанию
  name = information_schema,
  user = root,
  host = localhost,
  pass = "".

 статический getInstance() - подключение к базе данных.
